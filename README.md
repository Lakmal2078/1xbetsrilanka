# 1xBet Fast Cash

A promotional landing page for 1xBet Fast Cash — Sri Lanka's VIP Cash Agent service for fast deposits and withdrawals on 1xBet. The page is written in Sinhala (සිංහල) with English labels.

## Tech Stack

- **TanStack Start** (React SSR framework)
- **Tailwind CSS v4** for styling
- **Font Awesome 6** for icons
- **Netlify** for hosting and deployment

## Running Locally

```bash
npm install
npm run dev
```

The dev server starts on [http://localhost:3000](http://localhost:3000).
